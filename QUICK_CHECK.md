## 🎬 Сценарий демонстрации (5 минут)

### Минута 1: Главная
1. Показать баннер
2. Показать категории
3. Кликнуть "iPhone"

### Минута 2: Каталог
1. Показать фильтры
2. Применить фильтр по цене
3. Показать карточку товара

### Минута 3: Товар
1. Открыть iPhone 17 Pro
2. Выбрать цвет "Cosmic Orange" → картинка меняется
3. Добавить в корзину

### Минута 4: Корзина
1. Показать товар в корзине
2. Изменить количество
3. Заполнить форму (не отправлять)

### Минута 5: Админка
1. Войти в админку
2. Показать список товаров
3. Показать заявки
4. Показать создание товара (автозаполнение цветов)

---

## 🔥 Что показать обязательно

1. **Смена изображения при выборе цвета** - это уникальная фича
2. **Автозаполнение цветовых кодов** в админке
3. **Защита админ-панели** - попробовать зайти без авторизации
4. **Адаптивность** - показать на мобильном (F12 → Device Mode)
5. **Формы работают** - отправить хотя бы одну заявку

---

## ⚠️ Что НЕ показывать

1. Восстановление пароля (email не отправляется)
2. Карту на странице контактов (placeholder)
3. Оплату (не реализована)

## 🛠️ Если что-то сломалось

### Сайт не запускается
```bash
npm install
npm run dev
```

### База данных не работает
```bash
npx prisma db push
npx prisma db seed
```

### Нет товаров
```bash
npx tsx scripts/seed-products.ts
```

### Нет администратора
```sql
-- В Railway Dashboard → PostgreSQL → Query
UPDATE "User" SET role = 'ADMIN' WHERE email = 'admin@techstore.ru';
```
