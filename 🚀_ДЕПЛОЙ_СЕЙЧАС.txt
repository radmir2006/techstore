📋 СЛЕДУЮЩИЕ ШАГИ (5 минут):

1️⃣ ЗАГРУЗИТЕ КОД НА GITHUB
   
   git add .
   git commit -m "Ready for Vercel deployment"
   git push

   Если репозиторий не создан - создайте на github.com

═══════════════════════════════════════════════════════════════

2️⃣ ДЕПЛОЙ НА VERCEL

   1. Зайдите на https://vercel.com
   2. Войдите через GitHub
   3. Нажмите "Add New Project"
   4. Выберите ваш репозиторий
   5. Vercel автоматически определит Next.js

═══════════════════════════════════════════════════════════════

3️⃣ ДОБАВЬТЕ ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ

   Откройте файл: .env.vercel
   
   Скопируйте все переменные в Vercel:
   Settings → Environment Variables

   ВАЖНО: После первого деплоя замените URL!

═══════════════════════════════════════════════════════════════

4️⃣ НАЖМИТЕ DEPLOY

   Дождитесь завершения сборки (2-3 минуты)

═══════════════════════════════════════════════════════════════

5️⃣ ОБНОВИТЕ URL

   После деплоя:
   1. Скопируйте URL вашего сайта
   2. Обновите в Vercel переменные:
      - NEXTAUTH_URL
      - NEXT_PUBLIC_SITE_URL
   3. Нажмите "Redeploy"
