# 🚨 СРОЧНОЕ ИСПРАВЛЕНИЕ - Применение миграции

## Проблема
Сайт показывает ошибку "Application error: a server-side exception has occurred" потому что в коде используется поле `inStock`, но оно не существует в базе данных.

## ✅ Решение (выполните СЕЙЧАС)

### Шаг 1: Откройте Railway
1. Перейдите на https://railway.app
2. Откройте ваш проект
3. Найдите сервис **Postgres** (база данных)

### Шаг 2: Выполните SQL
1. Нажмите на сервис **Postgres**
2. Перейдите в раздел **Data**
3. Нажмите **Query** (или найдите поле для ввода SQL)
4. Скопируйте и вставьте этот SQL:

```sql
ALTER TABLE "Product" ADD COLUMN IF NOT EXISTS "inStock" BOOLEAN NOT NULL DEFAULT true;
UPDATE "Product" SET "inStock" = true;
CREATE INDEX IF NOT EXISTS "Product_inStock_idx" ON "Product"("inStock");
```

5. Нажмите **Execute** или **Run**

### Шаг 3: Проверьте сайт
1. Подождите 30 секунд
2. Откройте сайт: https://techstore-production-aa2a.up.railway.app/
3. Сайт должен заработать!

## Если не получается выполнить SQL

### Альтернатива: Через Shell
1. В Railway откройте сервис **techstore** (не Postgres!)
2. Нажмите три точки (...) → **Shell**
3. Выполните команду:
```bash
npx prisma db push --accept-data-loss
```
4. Дождитесь завершения (может занять 1-2 минуты)
