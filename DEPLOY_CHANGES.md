# 🚀 Деплой изменений на Railway

## ✅ Что было исправлено:

### 1. Картинка iPhone в категориях
**Проблема:** Изображение обрезалось и отображалось некорректно  
**Решение:** Изменен `object-cover` на `object-contain` с отступами

**Файл:** `components/CategoryGrid.tsx`
```tsx
// Было:
className="... object-cover"

// Стало:
className="... object-contain p-4"
```

### 2. Популярные товары подняты выше
**Проблема:** Секция "Популярные товары" была слишком низко  
**Решение:** Перемещена сразу после категорий

**Файл:** `app/page.tsx`
```
Порядок секций:
1. Баннер
2. Категории
3. Популярные товары ⬆️ (было ниже)
4. Trade-in
5. Новинки
6. IMEI Check
7. Преимущества
```

### 3. Кнопки "В корзину" на одной высоте ⭐ НОВОЕ
**Проблема:** Кнопки были на разной высоте из-за зачеркнутой цены  
**Решение:** Зарезервировано фиксированное место для старой цены

**Файл:** `components/ProductCard.tsx`
```tsx
// Зарезервировано место для старой цены (h-5)
// Если старой цены нет - показывается невидимый placeholder
<div className="text-sm h-5">
  {hasDiscount ? (
    <span className="text-gray-400 line-through">
      {formatPrice(product.oldPrice!)}
    </span>
  ) : (
    <span className="invisible">placeholder</span>
  )}
</div>
```

**Результат:**
- ✅ Все кнопки "В корзину" на одной высоте
- ✅ Зачеркнутая цена показывается только у товаров со скидкой
- ✅ Карточки выглядят аккуратно и единообразно

## 📤 Как задеплоить изменения:

### Вариант 1: Через Git (рекомендуется)

```bash
# 1. Добавить изменения
git add .

# 2. Создать коммит
git commit -m "fix: исправлено отображение iPhone в категориях, популярные товары подняты выше"

# 3. Отправить на GitHub
git push origin main
```

Railway автоматически задеплоит изменения через 1-2 минуты.

---

### Вариант 2: Через Railway CLI

```bash
# 1. Установить Railway CLI (если еще не установлен)
npm i -g @railway/cli

# 2. Войти в Railway
railway login

# 3. Связать проект
railway link

# 4. Задеплоить
railway up
```

---

### Вариант 3: Через Railway Dashboard

1. Откройте https://railway.app/dashboard
2. Найдите проект TechStore
3. Перейдите в раздел "Deployments"
4. Нажмите "Deploy" → "Redeploy"
