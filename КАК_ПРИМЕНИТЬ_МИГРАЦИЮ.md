# Как применить миграцию к production базе

## Способ 1: Через локальный скрипт (РЕКОМЕНДУЕТСЯ)

### Шаг 1: Получите DATABASE_URL из Railway

1. Откройте https://railway.app
2. Выберите ваш проект
3. Нажмите на сервис **Postgres**
4. Перейдите в раздел **Variables**
5. Найдите переменную **DATABASE_URL**
6. Скопируйте её значение (начинается с `postgresql://...`)

### Шаг 2: Установите переменную окружения

**Windows (PowerShell):**
```powershell
$env:DATABASE_URL="postgresql://postgres:пароль@хост:порт/railway"
```

**Windows (CMD):**
```cmd
set DATABASE_URL=postgresql://postgres:пароль@хост:порт/railway
```

### Шаг 3: Запустите миграцию

```bash
node migrate-production.js
```

Вы увидите:
```
🚀 Начинаем миграцию...
✅ Подключение установлено
✅ Колонка inStock добавлена
✅ Значения обновлены
✅ Индекс создан
✅ Миграция успешно применена!
```

## Способ 2: Через Railway Shell

1. Откройте https://railway.app
2. Выберите ваш проект
3. Нажмите на сервис **techstore** (не Postgres!)
4. Нажмите три точки (...) → **Shell**
5. Выполните команду:

```bash
npx prisma db push --accept-data-loss
```

---

## Способ 3: Через Railway SQL Query

1. Откройте https://railway.app
2. Выберите ваш проект
3. Нажмите на сервис **Postgres**
4. Перейдите в раздел **Data**
5. Нажмите **Query**
6. Вставьте и выполните:

```sql
ALTER TABLE "Product" ADD COLUMN IF NOT EXISTS "inStock" BOOLEAN NOT NULL DEFAULT true;
UPDATE "Product" SET "inStock" = true;
CREATE INDEX IF NOT EXISTS "Product_inStock_idx" ON "Product"("inStock");
```

## Проверка

После применения миграции:

1. Откройте сайт: https://techstore-production-aa2a.up.railway.app/
2. Сайт должен работать без ошибок!
3. Зайдите в админку: https://techstore-production-aa2a.up.railway.app/admin/login
4. Откройте любой товар на редактирование
5. Должна появиться галочка "В наличии" ✅

## Если возникли проблемы

Проверьте логи Railway:
1. Откройте проект на Railway
2. Выберите сервис **techstore**
3. Перейдите в раздел **Deployments**
4. Откройте последний деплой
5. Посмотрите логи на наличие ошибок
