## 🎯 О проекте

**TechStore** - это современный интернет-магазин техники Apple, разработанный как дипломный проект. Сайт включает полный функционал e-commerce платформы с админ-панелью для управления товарами и заказами.

### Основные возможности

#### Для покупателей:
- 📱 Каталог товаров с фильтрами и сортировкой
- 🛒 Корзина с оформлением заказа
- ❤️ Избранное
- ⚖️ Сравнение товаров (до 4 шт)
- 👤 Личный кабинет
- 📦 История заказов
- 🔄 Trade-in программа
- 🔍 Проверка IMEI
- 💬 Форма обратной связи

#### Для администраторов:
- 📊 Дашборд с статистикой
- 📦 Управление товарами
- 📋 Управление заявками
- 👥 Управление пользователями
- ⚙️ Настройки сайта
- 🎨 Управление баннерами

## 🛠️ Технологии

### Frontend
- **Next.js 14** - React фреймворк с App Router
- **TypeScript** - Типизация
- **Tailwind CSS** - Стилизация
- **Lucide React** - Иконки
- **React Hot Toast** - Уведомления

### Backend
- **Next.js API Routes** - Серверная логика
- **Prisma ORM** - Работа с БД
- **PostgreSQL** - База данных
- **NextAuth.js** - Аутентификация
- **Zod** - Валидация данных

### Деплой
- **Railway** - Хостинг
- **Vercel** - Альтернативный хостинг

## 🚀 Быстрый запуск

### Локально

```bash
# Установка зависимостей
npm install

# Настройка БД
npx prisma db push
npx prisma db seed

# Запуск
npm run dev
```

Сайт откроется на `http://localhost:3000`

### На Railway

1. Создайте проект на Railway

## 🔒 Безопасность

### Реализовано:
- ✅ Cookie-based аутентификация для админки

- ✅ Honeypot защита от ботов
- ✅ Валидация всех входных данных (Zod)
- ✅ Защита API маршрутов
- ✅ CSRF защита (NextAuth)
- ✅ SQL injection защита (Prisma)

## 🎯 Ключевые особенности

### 1. Синхронизация изображений с цветом
При выборе цвета товара автоматически меняется изображение в галерее.

### 2. Автозаполнение цветовых кодов
В админке при вводе названия цвета автоматически подставляется RGB код.

### 3. Защита админ-панели
Многоуровневая защита: middleware + layout + API проверка.

### 4. Rate Limiting
Защита от спама на всех формах обратной связи.

### 5. Адаптивный дизайн
Полностью адаптивный интерфейс для всех устройств.

### Возможные вопросы:

**Q: Почему выбрали Next.js?**  
A: Next.js предоставляет SSR, API Routes, оптимизацию изображений и отличный DX.

**Q: Как обеспечивается безопасность?**  
A: Cookie-based auth, rate limiting, валидация данных, защита API.

**Q: Масштабируемость?**  
A: Prisma ORM, PostgreSQL, модульная архитектура, API-first подход.

**Q: Почему PostgreSQL?**  
A: Надежность, ACID, поддержка сложных запросов, бесплатный хостинг на Railway.
