# 🧪 Руководство по тестированию TechStore

Это руководство поможет вам протестировать все функции сайта перед презентацией.

---

## 🚀 Быстрый старт

### Локальный запуск
```bash
npm install
npm run dev
```
Сайт откроется на `http://localhost:3000`

### Проверка сборки
```bash
npm run build
```
Должно завершиться без ошибок.

---

## 📋 Чек-лист тестирования

### 1. Главная страница (/)

**Что проверить:**
- [ ] Баннер отображается с изображением iPhone 17 Pro Max
- [ ] Кнопка "Купить iPhone" ведет в каталог iPhone
- [ ] Категории отображаются (iPhone, iPad, Mac, AirPods, Apple Watch)
- [ ] Клик по категории ведет в соответствующий каталог
- [ ] Новинки отображаются
- [ ] Header и Footer загружаются

**Как проверить:**
1. Откройте главную страницу
2. Кликните на баннер → должен перейти в `/catalog/iphone`
3. Кликните на категорию iPad → должен перейти в `/catalog/ipad`
4. Прокрутите вниз → проверьте отображение товаров

---

### 2. Каталог (/catalog/[slug])

**Что проверить:**
- [ ] Фильтры работают (бренды, цена, память, цвет)
- [ ] Сортировка работает (по цене, популярности, новизне)
- [ ] Переключение вида (сетка/список)
- [ ] Кнопка "В корзину" добавляет товар
- [ ] Кнопка "В избранное" работает (требует авторизации)
- [ ] Кнопка "Сравнить" работает (требует авторизации)

**Как проверить:**
1. Откройте `/catalog/iphone`
2. Примените фильтр по цене: 50000-100000
3. Выберите сортировку "Сначала дешевле"
4. Кликните "В корзину" на любом товаре
5. Проверьте счетчик корзины в Header (должен увеличиться)

**Тест фильтров:**
```
1. Выберите бренд "Apple"
2. Установите цену от 50000 до 150000
3. Выберите память "256GB"
4. Нажмите "Применить фильтры"
→ Должны отобразиться только подходящие товары
```

### 4. Корзина (/cart)

**Что проверить:**
- [ ] Товары отображаются
- [ ] Кнопки +/- изменяют количество
- [ ] Кнопка удаления работает
- [ ] Промокод применяется
- [ ] Форма оформления валидируется
- [ ] Заказ отправляется

**Как проверить:**
1. Добавьте 2-3 товара в корзину
2. Откройте `/cart`
3. Нажмите "+" на первом товаре → количество увеличится
4. Нажмите "−" → количество уменьшится
5. Нажмите "✕" → товар удалится

**Тест промокода:**
```
1. Введите промокод: SALE10
2. Нажмите "Применить"
→ Должна появиться скидка
```

**Тест оформления заказа:**
```
Имя: Петр Петров
Телефон: +7 (999) 888-77-66
Email: test@example.com (опционально)
Комментарий: Доставка после 18:00
✓ Согласие с политикой
→ Нажать "Оформить заказ"
→ Должен перейти на /cart/success
```

### 13. Админ-панель (/admin)

**Требуется роль ADMIN!**

#### Вход в админку (/admin/login)

**Тест входа:**
```
Email: admin@techstore.ru (или любой пользователь с ролью ADMIN)
Пароль: admin123
→ Нажать "Войти"
→ Должен перейти в /admin
```

**Создание администратора:**
1. Войдите как существующий админ
2. Откройте `/admin/users`
3. Найдите пользователя
4. Нажмите "Сделать администратором"
5. Выйдите и войдите под этим пользователем

---

#### Управление товарами (/admin/products)

**Создание товара (/admin/products/new):**
```
Название: iPhone 15 Pro Max
Slug: iphone-15-pro-max
Описание: Флагманский смартфон Apple
Категория: iPhone
Бренд: Apple
Цена: 129990
Старая цена: 139990 (опционально)
Память: 256GB (выбрать из dropdown)
Цвет: Black (автоматически заполнится код #000000)
Код цвета: #000000
SIM: SIM+eSIM
Тип варианта: color
Загрузить изображение
✓ Активен
✓ Новинка
```

**Редактирование товара:**
1. Откройте `/admin/products`
2. Нажмите "Редактировать" на любом товаре
3. Измените цену
4. Нажмите "Сохранить изменения"

**Удаление товара:**
1. Откройте `/admin/products`
2. Нажмите "Удалить" на товаре
3. Подтвердите удаление

## 🔍 Проверка безопасности

### 1. Защита админ-панели
```
1. Откройте /admin (не авторизованы)
→ Должен перенаправить на /admin/login

2. Попробуйте открыть /admin/products напрямую
→ Должен перенаправить на /admin/login

3. Войдите как обычный пользователь (не ADMIN)
4. Попробуйте открыть /admin
→ Должен показать ошибку или перенаправить
```

### 2. Rate Limiting
```
1. Откройте форму контактов
2. Отправьте 4 заявки подряд
→ 4-я заявка должна вернуть ошибку "Слишком много запросов"
```

### 3. Валидация форм
```
1. Попробуйте отправить форму с пустыми полями
→ Должны появиться ошибки валидации

2. Введите некорректный email
→ Должна появиться ошибка "Некорректный email"

3. Введите короткий телефон (меньше 10 цифр)
→ Должна появиться ошибка "Введите корректный номер"
```

## 🐛 Известные проблемы

### Не являются ошибками:
1. **"Проверьте почту"** при восстановлении пароля
   - Это только UI, реальная отправка email не настроена
   
2. **Placeholder карты** на странице контактов
   - Интеграция с картами не реализована

3. **Промокоды могут не работать**
   - Нужно создать промокоды в админке или БД

## 🎯 Рекомендуемый порядок демонстрации

1. **Главная** (30 сек) - баннер, категории
2. **Каталог** (1 мин) - фильтры, карточки товаров
3. **Товар** (1 мин) - выбор цвета, добавление в корзину
4. **Корзина** (1 мин) - оформление заказа
5. **Админка** (2 мин) - просмотр заявки, создание товара

**Общее время:** ~5-6 минут
