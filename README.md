# TechStore

E-commerce platform built with Next.js 14

## Tech Stack

Next.js 14, React, TypeScript, Tailwind CSS, PostgreSQL, Prisma

## Features

- Product catalog with filters
- Shopping cart
- Admin panel
- Responsive design

## Setup

npm install
npx prisma generate
npm run build
npm start

## Deploy

Deploy on Railway with GitHub integration
